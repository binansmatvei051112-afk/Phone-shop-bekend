# Phone Shop Backend 📱

Это серверная часть проекта «Магазин телефонов», написанная на **FastAPI**.

## Технологии
* **Python 3.x**
* **FastAPI** — для создания API.
* **SQLite** — база данных для хранения товаров (`Datasets.db`).
* **Git** — контроль версий.

## Структура проекта
* `main.py` — основной файл сервера.
* `models.py` — описание таблиц базы данных.
* `database.py` — логика подключения к БД.
* `backup.sql` — текстовый дамп базы данных для восстановления.

## Как запустить
1. Активировать виртуальное окружение:
   ```powershell
   .venv\Scripts\activate
2. Запустить сервер:
    uvicorn main:app --reload
